Description

The goal of this project was to focus on hands-on practice with a modern human-centered design process while building a web application to serve a set of target users. The way we materialized this goal was via an web application concept that aims to ease the shopping experience in its entirety by eliminating the need for users to conduct research on a product.

Responsibility

My main role was to implement mockups into user testable prototypes and in turn, present these to an audience. I shared the responsibility of performing user testing and turning feedback into thoughtful design changes.
